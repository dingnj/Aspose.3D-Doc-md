## **Download von NPM**
Aspose.3D für Node.js via Java Beispiele und Projektpakete sind auf [NPM](https://www.npmjs.com/package/aspose.3d) verfügbar. Sie können den Befehl `npm i aspose.3d` verwenden, um es herunterzuladen. Sie können auch die ZIP-Datei von [hier](https://releases.aspose.com/3d/nodejs-java/) herunterladen.

Entpacken Sie den Inhalt der ZIP-Datei in einen beliebigen Ordner auf Ihrem Computer. Sie können das Projektpaket einsehen.

Die `README`-Datei enthält einige Beispielcode.

{{% alert color="primary" %}} 

Öffnen Sie das Projekt mit VSCode und befolgen Sie die Schritte. Sie sollten nun in der Lage sein, die Beispiele normal auszuführen.

{{% /alert %}} 
## **Führen Sie das Beispiel aus**
Nachdem Sie das Projekt in VSCode geöffnet haben, finden Sie die README-Datei, wählen Sie ein Beispiel daraus aus und kopieren Sie es in die Datei index.js. Wenn der Pfad der aspose.threed-Datei falsch ist, müssen Sie ihn ändern. Geben Sie den Befehl `node index.js` im Terminal ein. Dann sehen Sie die erfolgreich generierte 3D-Datei im Ordner.
